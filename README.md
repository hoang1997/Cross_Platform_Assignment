# Cross_Platform_Assignment
Cross platform assignment using phonegap, jQuery mobile, Cordova, and Cordova Plugins 

API's used:

- Edamam Recipe API
- Barcode Lookup API
- Google Maps API
    - Geolocation
    - Places

Plugins used:

- phonegap-barcode-scanner
- cordova-plugin-camera
